# Finance-LangChain

A minimal LangChain-based finance project.

## Setup

1. Create and activate a Python environment (optional but recommended).
2. Install dependencies:

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## Notes
- Ensure any required API keys are set via environment variables or a `.env` file if your code expects them.
- The `faiss_store_openai/` folder contains the FAISS index artifacts.
